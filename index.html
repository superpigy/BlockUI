Welcome to Cai8's Metro World

Block UI is an open source library for Microsoft 'Metro' UI lovers. With similar delegates and methods,the implementing of 'Block UI' is no more complex than UITableView.

Key delegates

- (BlockBox *)blockBoxForContainer:(BlockContainer *)container atIndex:(NSInteger)index;

- (CGSize)sizeForBoxForContainer:(BlockContainer *)container atIndex:(NSInteger)index;

- (CGFloat)marginBetweenBoxes;

- (NSInteger)numberOfBoxes;

UI defines and tweaks

All UI paramaters such as margin,coordinates,unit length are all defined in 'BlockUI.h', you can easily change them in order to satisfy your own style. (Many other features and UI modules will be added in future release)

#define DEFAULT_HORIZONTAL_UNITS
#define DEFAULT_VERTICAL_UNITS
#define DEFAULT_WIDTH_OF_UNIT
#define DEFAULT_HEIGHT_OF_UNIT
#define DEFAULT_MARGIN_OF_UNIT

#define GROUP_LEFT_BOUNDARY_MARGIN
#define GROUP_TOP_BOUNDARY_MARGIN

#define MARGIN_BETWEEN_GROUPS

#define MAXIMUM_BLOCK_CONTAINER_BUFFER

Installation

  • Drag 'BlockUI' folder to your project
  • Add 'BlockUI.h' to your .pch file
  • Add QuartzCore.framework
  • Done

Implementation

Import BlockContainer.h to your view or view controller and do some simple initialization

#import "BlockContainer.h"

@implementation YourController
- (void)viewDidLoad
{
     BlockContainer *container = [[BlockContainer alloc] initWithFrame:YourRect];
     [container setDataSource:self];
     [container setDelegate:self];
}
@end
